# Social Links Profile

A solution to the Social Links Profile challenge from Frontend Mentor.

## Overview

This project is a responsive social profile card that displays a user's information and links to their social platforms.  
The focus was on clean layout, proper spacing, and interactive hover states.

## The Challenge

Users should be able to:

- View the optimal layout depending on their device screen size
- See hover states for all interactive elements

## Built With

- HTML5
- CSS3
- Flexbox
- CSS Variables
- Mobile-first workflow

## Features

- Responsive centered layout
- Clean semantic HTML structure
- Fully clickable link blocks
- Smooth hover transitions
- Custom color system using CSS variables
- Accessible link structure

While working on this project, I practiced:
